MMRIA CSV Export Format

The MMRIA export function produces a zip file that contains one or more case data files plus a data dictionary. The zip file is downloaded directly to the “Downloads” folder in the local environment of the user’s computer.

MMRIA Export Case File(s) One or more files that contain MMRIA case data. Abbreviated MMRIA field names are created during the export process, and used as column headers for the data files. These abbreviated field names are suitable for use with external statistical applications.

MMRIA Export Data Dictionary An additional data-dictionary file is created when MMRIA case data are exported. The name of this file is: "data-dictionary.csv". Each data-dictionary file is specific to the MMRIA case data that is exported with. This file is used to map the abbreviated field names in the exported data file to the full path field names contained in the MMRIA database. File columns include the exported file name, the abbreviated export field name, the full MMRIA field name, and the MMRIA field prompt.
