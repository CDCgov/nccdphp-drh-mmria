Data Dictionary

The MMRIA export function produces a zip file that contains one or more case data files plus a data dictionary. The zip file is downloaded directly to the “Downloads” folder in the local environment of the user’s computer.

MMRIA Export Case Files

These files contain MMRIA case data. Abbreviated MMRIA field names are displayed as column names. These abbreviated column names meet the naming requirements of commonly used statistical applications (e.g., SAS).

 

MMRIA Export Data Dictionary

A data dictionary file is created when MMRIA case data are exported. The column names included in the data dictionary correspond to variable names when the data are imported into a standard statistical application. This file can be used to map the column names in the exported data files to the file and full path from where it came in the MMRIA database. For example, the column name “bfdcpr_ro_mothe” refers to the race field on the parent section of the Birth/Fetal Death Certificate form within MMRIA. Note that each data-dictionary file is specific to the MMRIA case data that it is exported with.
